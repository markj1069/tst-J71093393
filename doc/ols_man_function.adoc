== Exit Status

Here is a list of the exit codes that *{function_name}* can return, along with an explanation of each code. The exit code is a number between 0 and 255, where 0 indicates successful completion and any non-zero value indicates an error or warning.

[horizontal]

EX = 0:: Successful completion.

EX = 4:: Warning, but the function completed successfully.

EX = 8:: Error, but part of the function completed okay.

EX &ge; 16:: Fatal Error, the function did not complete successfully.
